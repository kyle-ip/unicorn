# Unicorn
[![Python](https://img.shields.io/badge/language-python-yellow.svg)]()

A Python toy framework for building web applications.

## Features

- WSGI interfaces
- URL routing
- View functions
- Session
- JSON interface
- Database Access Wrapper

## TODO

- Template Engine

## Usage

Python 3.5+ Required.

```shell
pip install -r requirements.txt
```

Run demo.

```shell
python runserver.py
```
